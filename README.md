# Invoicing System 
`work in progress `

Add contractors, products and create invoices in pdf format

# Running
1. Prepare .env file in the root of the project
```
NODE_ENV=production
JWT_SECRET=<secret>
```
2. Run application
```
$ docker-compose build && docker-compose up
```
Ports exposed:
- frontend: 3000
- backend: 5000
- mongodb: 27017
